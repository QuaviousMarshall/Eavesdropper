# 🤖 Мобильное приложение «Трон»

Android-приложение — **ИИ-ассистент на базе моделей OpenAI**, ориентированное на работу с голосовыми запросами пользователя и взаимодействие через уведомления. Проект разрабатывается как учебно‑практический, с акцентом на современный Android‑стек, чистую архитектуру и production‑подходы.

---

## 🧠 Суть приложения

«Трон» — это голосовой ИИ‑ассистент, который:
- работает **как в фокусе приложения, так и вне его**;
- **записывает голос пользователя**, пока нажата кнопка активации и приложение не закрыто;
- **распознаёт речь**, фиксирует вопросы в потоке речи;
- формирует текстовый запрос и **отправляет его в OpenAI API**;
- получает **краткий ответ** от модели;
- доставляет результат пользователю через **системные уведомления**.

Ответы отображаются с использованием `NotificationChannel` и `NotificationManager`.

---

## ✨ Основные возможности

- Голосовой ввод и обработка пользовательской речи  
- Формирование текстовых запросов из речи  
- Работа с OpenAI API  
- Отправка ответов через системные уведомления  
- Работа приложения в фоне  
- Аутентификация пользователей через Firebase  
- Экран профиля и экран настроек  

---

## 🏗 Архитектура

Проект строится на принципах **Clean Architecture**:
- разделение на `data / domain / presentation` слои;
- бизнес‑логика изолирована от UI;
- высокая тестируемость и расширяемость.

В presentation‑слое используется **MVVM**:
- `ViewModel`;
- `StateFlow`;
- интеграция с `Lifecycle`.

Для внедрения зависимостей применяется **Dependency Injection**.

---

## 🌐 Работа с сетью

Приложение активно использует сетевое взаимодействие:

- работа с REST‑сервисами;
- обработка асинхронных потоков данных;
- централизованная обработка ошибок.
